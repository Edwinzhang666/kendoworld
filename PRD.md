# PRD — 《一介新手，制霸剑道全国》

概要
- 定位：单机卡牌通关 + 可扩展 PVP/多人，短局策略 + 强叙事，适配微信小程序
- 调性：搞笑 / 戏谑 / 中二 / 剑道圈讽刺
- 目标：实现可运行的小程序 MVP（卡牌引擎、事件系统、比赛与积分流程）并上线

核心模块
- 卡牌引擎（Card Engine）
- 事件系统（Event Manager）
- 比赛引擎（Match Engine）
- 成长 & 积分（Progression / Ranking）
- UI（小程序） + 存储（本地/云）

初始创建与数值要点
- 城市：一线/二线/小城市（影响金币、对手强度、曝光系数）
- 道馆：魔鬼/佛系/技术/资本/老派（影响成长、伤病率、卡池）
- 流派：中段/上段/二刀（专属卡池、事件）

数据与模块化实现
- 卡牌与事件数据以 JSON 存储（cards.json / events.json）
- 卡牌字段：id,name,type,cost,effect,rarity,growth
- 事件模板：id,title,options,effects

小程序实现要点
- 技术栈建议：Taro + React + TypeScript（推荐）
- 存储：wx.setStorageSync 本地进度 + 云 DB（排行榜/云存档可选）
- CI：GitHub Actions -> 构建 -> 微信云构建 / 手动上传

MVP 范围（可交付）
1. 卡牌引擎（回合制基础，可抽牌、出牌、消耗计算）
2. 事件系统（几类随机/剧情事件）
3. 单人比赛（AI 对手，积分结算）
4. 初始创建界面与卡组编辑
5. 示例卡牌/事件集

发布与合规
- 游戏归类为“互动游戏”，提交时准备隐私政策与游戏说明

—— End PRD ——
